### Iniciar Proyecto GIT

    $ git init

### Saber el estado del respositorio

    $ git status

### Preparar archivos (unstage)

    $ git add <file>
    $ git add -A
    $ git add .
    $ git add *

### Guardar archivos en el repositorio

    # me abre un editor
    $ git commit 
    # guardo automaticamente con un mensaje
